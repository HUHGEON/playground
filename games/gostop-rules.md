# 화투(맞고/고스톱) 룰 명세 — 구현 계약서

> 이 문서가 `games/gostop.js` 구현의 **단일 기준(source of truth)**. 코드와 어긋나면 이 문서를 따른다.
> ⚙️ = config로 빼는 하우스룰(§7).

## 0. 모드 파라미터
| 항목 | 맞고(2인) | 고스톱(3인) |
|---|---|---|
| 손패 | 10장 | 7장 |
| 바닥패 | 8장 | 6장 |
| 나는 최소 점수 | 7점 | 3점 |
| 피박 기준 | 진 사람 피 ≤ 7장 | 진 사람 피 ≤ 5장 |
| 정산 | 진 사람이 점수×배율 지불 | 진 사람들 각자 지불 |
| 고박 | 없음 | 있음 |
| 독박 | 없음 | 있음 |
| 광팔기 | 없음 | 4인+일 때만 |
| 멍박 | 옵션(off 기본) | 옵션(on 기본) |
| 자뻑 | 쪽 없이도 성립 | 쪽 성립 시에만 |

- 3인째 입장 → **다음 판부터** 맞고→고스톱 전환. 4인은 한 명 광팔기(후속 단계).

## 1. 카드 데이터 모델
총 **51장** = 화투 48 + 보너스피 3.
카드 속성: `{ month: 1~12, category, isBonus, piValue, flags[] }`
- `category`: `KWANG`(광) | `YEOL`(열끗) | `TTI`(띠) | `PI`(피)
- `piValue`: 일반피=1, 쌍피=2, 쓰리피=3, 나머지=0
- `flags`: `BIGWANG`(비광) | `SSANGPI`(쌍피) | `GODORI`(고도리 새) | `BONUS`

### 월별 구성 (4장 × 12월 = 48)
| 월 | 카드1 | 카드2 | 카드3 | 카드4 |
|---|---|---|---|---|
| 1 송학 | 광 | 홍단 | 피 | 피 |
| 2 매조 | 열끗(고도리) | 홍단 | 피 | 피 |
| 3 벚꽃 | 광 | 홍단 | 피 | 피 |
| 4 흑싸리 | 열끗(고도리) | 초단 | 피 | 피 |
| 5 난초 | 열끗 | 초단 | 피 | 피 |
| 6 모란 | 열끗 | 청단 | 피 | 피 |
| 7 홍싸리 | 열끗 | 초단 | 피 | 피 |
| 8 공산 | 광 | 열끗(고도리) | 피 | 피 |
| 9 국진 | 열끗⇆쌍피 | 청단 | 피 | 피 |
| 10 단풍 | 열끗 | 청단 | 피 | 피 |
| 11 오동 | 광 | 쌍피 | 피 | 피 |
| 12 비 | 비광 | 열끗 | 비단(띠) | 쌍피 |

- 광 5: 1·3·8·11·12(비광) / 열끗 9: 2·4·5·6·7·8·9·10·12 (고도리=2·4·8)
- 띠 10: 홍단(1,2,3)·청단(6,9,10)·초단(4,5,7)·비단(12) / 피 24(쌍피 2 = 11·12월)
- **9월 국진**: 열끗 OR 쌍피 중 **유리한 한쪽으로만** 카운트(동시 중복 금지). 정산 시 보유자에게 유리한 쪽 자동 선택.

### 보너스피 3장 (48 밖, isBonus)
- 기본 구성: 쌍피2 + 쓰리피1 (piValue 2,2,3). ⚙️ `bonusCardCount` 2|3.
- 동작 §3.4.

### 에셋 (이미 슬라이스됨)
- 월카드: `public/gostop/{월}-{0~3}.png` (카드1=인덱스0 … 카드4=인덱스3, 위 표 순서)
- 보너스: `public/gostop/plus.svg` (+ 쌍피/쓰리피 변형 추가 예정)

## 2. 셋업
- **선**: 첫 판 랜덤, 이후 직전 승자. ⚙️ `seonAfterNagari` keep|rotate.
- 딜링: 모드별 손/바닥 장수 분배.
- 셋업 예외:
  - 바닥 같은 월 3장 → 뻑 예비(묶음) 표시.
  - 바닥 같은 월 4장 → ⚙️ 재섞기(기본) | 선이 가져감.
  - 바닥 보너스피 → 선이 가져감.
  - 손패 같은 월 4장(총통) → §6.

## 3. 턴 진행 (선부터 반시계)
한 턴 = ①손패1장 → ②바닥매칭 → ③더미1장뒤집기 → ④바닥매칭 → ⑤이벤트판정 → ⑥획득정리

### 3.1 기본 매칭
- 낸 패와 같은 월 바닥에 있으면 둘 다 먹음.
- 같은 월 바닥 2장이면 ⚙️ `floorMatchChoice` auto(1장)|player.
- 없으면 바닥에 깐다.

### 3.2 이벤트 판정 (우선순위 순)
| 이벤트 | 조건 | 효과 |
|---|---|---|
| 쪽 | 짝 없어 깐 패에, 뒤집은 패가 같은 월 | 둘 다 먹기 + 상대 피 1 |
| 따닥 | 손패 1쌍 + 뒤집은 패로 또 다른 1쌍(같은 월 4장 동시) | 4장 먹기 + 상대 피 1 |
| 싹쓸이 | 매칭 후 바닥 완전히 빔 | 상대 피 1 (중첩 누적) |
| 뻑 | 바닥 같은 월 2장 위 내 패까지 3장 못 먹음 | 못 먹음. 다음에 그 월 먹는 사람이 상대 피 1 |
| 자뻑 | 낸 패+뒤집은 패로 내가 만든 뻑을 같은 턴에 회수 | 피 2장(효자뻑 ⚙️) |
| 첫뻑 | 첫 턴에 뻑 | 5점 상당 ⚙️ |
| 연뻑 | 첫뻑 다음 턴 또 뻑 | 10점 / 삼연뻑 20점 ⚙️ |

- 피 뺏기 예외: ①마지막 턴엔 안 줌 ②피 0장 상대에겐 못 뺏음 ③상대가 쌍피만 있으면 쌍피를 뺏음.

### 3.3 흔들기 / 폭탄
- **흔들기**: 손에 같은 월 3장, 자기 턴에 선언+공개 → 그 판 ×2(두 묶음 ×4).
- **폭탄**: 흔든 월 4번째가 바닥에 깔리면 3장 투하 → 그 월 전부 + 상대 피 1 + 손패 2장 부족분만큼 "안 내고 뒤집기만" 2턴.
- ⚙️ 콩알탄(2장 폭탄): 뒤집기만 1턴, 흔들기 배율 미반영.

### 3.4 보너스피
- 손패에서 내거나 더미에서 뒤집히면: ①즉시 내 피더미(piValue 가산) ②더미 한 장 더 뒤집기(또 보너스면 반복).
- ⚙️ `bonusStealPi`(기본 on): 보너스 획득 시 상대 피 1.
- 뒤집은 보너스 직후 뻑이면 보너스도 뻑더미에 붙음(⚙️ 지역차).

## 4. 점수 족보
| 족보 | 규칙 |
|---|---|
| 광 | 3광 3점 / 비광 낀 3광 2점 / 4광 4점(비광 껴도) / 5광 15점 |
| 고도리 | 2·4·8 열끗 전부 = 5점 |
| 열끗 | 5장 1점, +1장당 +1 |
| 멍박(멍따) | 열끗 7장+ → 최종 ×2(배수) |
| 띠 | 5장 1점, +1장당 +1 |
| 홍/청/초단 | 각 3장 완성 시 각 3점 |
| 피 | 10장 1점, +1장당 +1 (쌍피·보너스 piValue=2) |

- ⚙️ `bidanIsChodan`: 비단 초단 인정(기본 미인정). 단 띠 5장 카운트엔 비단 포함.
- ⚙️ 구사(9월 열끗3)·비도리 등 변형 미적용 기본.

## 5. 고/스톱 & 정산
### 5.1 규칙
- 최소점수(3/7) 도달 시 매 턴 고|스톱.
- 고 선언 시: 직전 고 점수보다 최소 +1점 더 나야 다음 고/스톱.
- 고 후 피 뺏겨 깎이면 보충해야 다시 선언.

### 5.2 정산 계산 순서 (★엄수)
```
1) baseScore = 광+고도리+열끗+띠+단+피
2) goScore   = base + 고점수
   - 1고:+1, 2고:+1(누적)
   - 3고부터: 누적점 ×2씩 (3고×2, 4고×4, 5고×8…)
3) multiplier(곱셈): 흔들기×2(묶음당) · 멍박×2 · 패자별 박×2 각각(피박+광박→×4)
4) 나가리 누적배수 ×2^(누적횟수) 최종 곱
```

### 5.3 박 (×2 중첩)
| 박 | 조건 | 효과 |
|---|---|---|
| 피박 | 승자 피점수로 났고 + 진사람 피 ≤5(맞고7)장(0장 제외) | 그 패자 ×2 |
| 광박 | 승자 광점수(3+)로 났고 + 진사람 광 0장 | 그 패자 ×2 |
| 멍박 | 승자 열끗 7장+ | 전체 ×2 |
| 고박(3인) | 고 외친 뒤 다른 사람이 먼저 남 | 고한 사람 전원분 전액 부담 |
| 독박(3인) | 아무도 고 안 한 상태서 상대가 던진 패로 광/홍단/청단/고도리/구사 완성해 남 | 그 패 던진 사람 전액(쌍피류·마지막턴 제외) |

## 6. 종료 조건
- **스톱**: 즉시 정산.
- **나가리**: 고한 사람이 추가점 못 내거나 패 소진까지 무승부 → 다음 판 ×2 누적.
- **쓰리뻑**: 한 사람 뻑 3번 → 즉시 승리(3/맞고7점). ⚙️ 진행 중에만, 마지막턴 제외.
- **총통**: 손패 같은 월 4장(또는 5광) → 즉시 승리(10점 ⚙️) 또는 거부 후 진행. ⚙️ 들고치기 인정.
