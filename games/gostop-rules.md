# 화투(맞고/고스톱) 룰 명세 — 구현 계약서

> 이 문서가 `games/gostop.js` 구현의 **단일 기준(source of truth)**. 코드와 어긋나면 이 문서를 따른다.
> ⚙️ = config로 빼는 하우스룰(§7).

## 0. 모드 파라미터
| 항목 | 맞고(2인) | 고스톱(3인) |
|---|---|---|
| 손패 | 10장 | 7장 |
| 바닥패 | 8장 | 6장 |
| 나는 최소 점수 | 7점 | 3점 |
| 피박 기준 | 진 사람 피 ≤ 7장 | 진 사람 피 ≤ 5장 |
| 정산 | 진 사람이 점수×배율 지불 | 진 사람들 각자 지불 |
| 고박 | 없음 | 있음 |
| 독박 | 없음 | 있음 |
| 광팔기 | 없음 | **없음**(4인도 다 참여) |
| 멍박 | 옵션(off 기본) | 옵션(on 기본) |

- 3인째 입장 → **다음 판부터** 맞고→고스톱 전환.
- **4인도 7장/6장**(3인과 동일, 광팔기 없음). 더미 소진되면 그 뒤 턴은 뒤집기 생략(손패만 냄).

## 1. 카드 데이터 모델
총 **51장** = 화투 48 + 보너스피 3.
카드 속성: `{ month: 1~12, category, isBonus, piValue, flags[] }`
- `category`: `KWANG`(광) | `YEOL`(열끗) | `TTI`(띠) | `PI`(피)
- `piValue`: 일반피=1, 쌍피=2, 쓰리피=3, 나머지=0
- `flags`: `BIGWANG`(비광) | `SSANGPI`(쌍피) | `GODORI`(고도리 새) | `BONUS`

|---|---|---|---|---|
| 1 송학 | 광 | 홍단 | 피 | 피 |
| 2 매조 | 열끗(고도리) | 홍단 | 피 | 피 |
| 3 벚꽃 | 광 | 홍단 | 피 | 피 |
| 4 흑싸리 | 열끗(고도리) | 초단 | 피 | 피 |
| 5 난초 | 열끗 | 초단 | 피 | 피 |
| 6 모란 | 열끗 | 청단 | 피 | 피 |
| 7 홍싸리 | 열끗 | 초단 | 피 | 피 |
| 8 공산 | 광 | 열끗(고도리) | 피 | 피 |

### 보너스피 3장 (48 밖, isBonus)
- 기본 구성: 쌍피2 + 쓰리피1 (piValue 2,2,3). ⚙️ `bonusCardCount` 2|3.
- 동작 §3.4.

### 에셋 (이미 슬라이스됨)
- 월카드: `public/gostop/{월}-{0~3}.png` (카드1=인덱스0 … 카드4=인덱스3, 위 표 순서)

## 2. 셋업
- **선**: 첫 판은 패 돌리고 바닥패 뒤집어 **가장 높은 패 뽑은 사람**, 이후 직전 승자. ⚙️ `seonAfterNagari` keep|rotate.
- 딜링: 모드별 손/바닥 장수 분배.
- 셋업 예외:
  - 바닥 같은 월 3장 → 뻑 예비(묶음) 표시.
  - 바닥 같은 월 4장 → ⚙️ 재섞기(기본) | 선이 가져감.
  - 바닥 보너스피 → 선이 가져감.
  - 손패 같은 월 4장(총통) → §6.

## 3. 턴 진행 (선부터 반시계)
한 턴 = ①손패1장 → ②바닥매칭 → ③더미1장뒤집기 → ④바닥매칭 → ⑤이벤트판정 → ⑥획득정리

### 3.1 기본 매칭
- 낸 패와 같은 월 바닥에 있으면 둘 다 먹음.
- 같은 월 바닥 2장이면 ⚙️ `floorMatchChoice` auto(1장)|player.
- 없으면 바닥에 깐다.

### 3.2 이벤트 판정 — ★손패를 바닥에 올린 뒤 "뒤집기와 합산" 판정
손패를 낸 월 M, 내기 전 바닥의 M 개수 cB, 뒤집은 패의 월 N 기준:

| 상황 | 조건 | 효과 |
|---|---|---|
| **쪽** | cB=0 & N=M (짝 없어 깐 패를 뒤집기가 같은 월) | 깐 패+뒤집기 먹기 + 상대 피 1 |
| **뻑** | cB=1 & N=M (1장 먹으려 냈는데 뒤집기 같은 월) | 3장 스택, **못 먹음**. 나중에 4번째 내는 사람이 전부 먹고 상대 피 1 |
| **따닥** | cB=2 & N=M (2장 위에 내고 뒤집기까지 같은 월=4장) | 4장 한번에 먹기 + 상대 피 1 |
| **뻑먹기** | 바닥 스택(3장)의 4번째를 냄/뒤집음 | 4장 먹기 + 상대 피 1 |
| **쓸(싹쓸이)** | 먹은 뒤 바닥이 완전히 빔 | 상대 피 1 (쪽/따닥과 **중첩** → 2장) |
| (cB=2 & N≠M) | 바닥 2장에 매칭 냄 | 낸 패 + 둘 중 **1장 선택**해 먹기(나머지 1장 남김). 뻑 아님 |
| (뒤집기 N이 바닥 2장과 매칭) | flip이 동월 2장 만남 | 1장 **선택**해 먹기(§3.1) |

- 첫뻑/연뻑·자뻑 점수는 ⚙️(`firstBbeokPoint`) — 후속.
- 피 뺏기 예외: ①자기 패 소진(마지막 턴) 안 줌 ②피 0장 상대 못 뺏음 ③쌍피만 있으면 쌍피를 뺏음.

### 3.3 흔들기 / 폭탄
- **흔들기**: 손에 같은 월 3장 → 자기 턴에 공개 선언 → 셋 중 1장을 내려놓음.
- **폭탄**: 손에 같은 월 3장 + 바닥에 그 월 나머지 1장 → 3장을 한번에 투하(그 월 4장 전부 회수).
- 흔들거나 폭탄한 사람이 **이긴 경우(스톱)** 1회마다 최종점수 ×2. (배수는 스톱 기준 적용)

### 3.4 보너스피
- **손패에서 낼 때**(플러스피 포함): ①즉시 내 피더미(piValue 가산) ②더미 한 장 **내 손으로 보충**(공짜) ③상대 피 1 뺏기 → **턴 소비 안 함**(한 장 더 냄). 보충 못해 손패 0장이면 정상 종료 흐름.
- **더미에서 뒤집히면**: ①내 피더미 ②더미 한 장 더 뒤집기(또 보너스면 반복) ③상대 피 1.
- ⚙️ `bonusStealPi`(기본 on): 보너스 획득 시 상대 피 1.
- 뒤집은 보너스 직후 뻑이면 보너스도 뻑더미에 붙음(⚙️ 지역차).

|---|---|
| 광 | 3광 3점 / 비광 낀 3광 2점 / 4광 4점(비광 껴도) / 5광 15점 |

- ⚙️ 구사(9월 열끗3)·비도리 등 변형 미적용 기본.

## 5. 고/스톱 & 정산
### 5.1 규칙
- 최소점수(3/7) 도달 시 매 턴 고|스톱.
- 고 선언 시: 직전 고 점수보다 최소 +1점 더 나야 다음 고/스톱.
- 고 후 피 뺏겨 깎이면 보충해야 다시 선언.

### 5.2 정산 계산 순서 (★엄수)
```
1) baseScore = 광+고도리+열끗+띠+단+피
2) goScore   = (base + goCount) × goMult
   - goMult = goCount<=2 ? 1 : 2^(goCount-2)
   - 1고:(base+1)    2고:(base+2)    3고:(base+3)×2
     4고:(base+4)×4  5고:(base+5)×8  6고:(base+6)×16
3) multiplier(곱셈): 흔들기/폭탄 ×2(1회마다) · 멍박×2 · 패자별 박×2 각각(피박+광박→×4)
   ※ 흔들기/폭탄·박 배수는 모두 "스톱(승리) 기준"으로만 적용
4) 나가리 누적배수 ×2^(누적횟수) 최종 곱
```

### 5.3 박 (×2 중첩)
| 박 | 조건 | 효과 |
|---|---|---|
| 피박 | 승자 피점수로 났고 + 진사람 피 ≤5(맞고7)장(0장 제외) | 그 패자 ×2 |
| 광박 | 승자 광점수(3+)로 났고 + 진사람 광 0장 | 그 패자 ×2 |
| 멍박 | 승자 열끗 7장+ | 전체 ×2 |
| 고박(3인) | 고 외친 뒤 다른 사람이 먼저 남 | 고한 사람 전원분 전액 부담 |
| 독박(3인) | 아무도 고 안 한 상태서 상대가 던진 패로 광/홍단/청단/고도리/구사 완성해 남 | 그 패 던진 사람 전액(쌍피류·마지막턴 제외) |

## 6. 종료 조건
- **스톱**: 즉시 정산.
- **나가리**: 고한 사람이 추가점 못 내거나 패 소진까지 무승부 → 다음 판 ×2 누적.
- **쓰리뻑**: 한 사람이 뻑 3번 → **자동 승리**(3점/맞고7점 상당).
- **총통**: 받은 손패에 같은 월 4장 → 패 공개하고 **즉시 끝내면 자동 3점 승**. 총통승 안 하고 계속하면 그 사람이 이겼을 때 **점수 ×4**.
